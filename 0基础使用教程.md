# QuantLife 0基础本地使用教程

这份教程写给不会 GitHub、不会命令行的用户。你只需要会下载、解压、双击文件。

## 一、你会得到什么

分享者通常会发给你一个压缩包，例如：

```text
QuantLife_Web_新手版.zip
```

你解压后会看到一个文件夹，里面有一个：

```text
启动 QuantLife.bat
```

以后使用 QuantLife，优先双击这个文件。

## 二、第一次使用前：安装 Node.js

QuantLife 是在你自己的电脑本地运行的网页工具，需要先安装 Node.js。

1. 打开 Node.js 官网：

```text
https://nodejs.org/
```

2. 下载 LTS 版本。
3. 一路点击安装即可。
4. 安装完成后，重启一下电脑更稳妥。

只需要安装一次。以后不用重复安装。

## 三、解压 QuantLife

1. 右键 `QuantLife_Web_新手版.zip`
2. 点击“全部解压”或“解压到当前文件夹”
3. 进入解压出来的文件夹

注意：不要直接在压缩包里双击启动，一定要先解压。

## 四、启动 QuantLife

在文件夹里找到：

```text
启动 QuantLife.bat
```

双击它。

第一次启动时，它会自动安装运行依赖，可能需要几分钟。请等待窗口继续运行，不要关闭。

启动成功后，浏览器会自动打开：

```text
http://127.0.0.1:3030
```

如果浏览器没有自动打开，你可以手动复制上面的地址到浏览器地址栏。

## 五、使用时不要关闭黑色窗口

双击启动后，会出现一个黑色窗口。

这个窗口就是 QuantLife 的本地服务。使用 QuantLife 时，请保持它打开。

如果你关闭了这个窗口，网页可能就不能继续保存和同步本地数据。下次使用时，再双击 `启动 QuantLife.bat` 即可。

## 六、设置自己的信息

第一次打开页面后，建议先进入“设置中心”：

- 修改昵称
- 修改头像
- 修改应用名称
- 修改主页标语
- 修改成长维度
- 修改每个维度的目标和经验系数

这些内容都是给你自己用的，后续也可以随时修改。

## 七、配置自己的 AI API

QuantLife 不会自带任何人的 API Key。

如果你只想手动打卡，可以先不配置 API。

如果你想使用 AI 文字解析、AI 规划等功能，需要在“设置中心”里填写你自己的 API：

- Provider
- Base URL
- Model / Endpoint
- API Key / Token

填写完成后点击保存，并测试连接。

注意：不要把自己的 API Key 发给别人。

## 八、数据保存在哪里

你的数据保存在当前文件夹里，常见文件包括：

```text
fengdingding-progress.json
fengdingding-progress.db
fengdingding-progress.db-shm
fengdingding-progress.db-wal
backups/
llm-config.json
```

如果你要备份自己的 QuantLife 数据，可以备份这些文件。

其中 `llm-config.json` 里可能包含你的 API Key，请不要分享给别人。

## 九、怎么把这个工具发给朋友

如果你是分享者，建议只分享新手版压缩包，不要分享自己的本地数据文件。

不要分享这些文件：

```text
.env
llm-config.json
fengdingding-progress.json
fengdingding-progress.db
fengdingding-progress.db-shm
fengdingding-progress.db-wal
backups/
node_modules/
```

安全做法：用项目里的 `打包新手版.ps1` 生成压缩包，再把生成的 ZIP 发给朋友。

## 十、常见问题

### 1. 双击后提示 Node.js is not installed

说明你的电脑还没有安装 Node.js。

请先去这里安装 LTS 版本：

```text
https://nodejs.org/
```

### 2. 第一次启动很慢

第一次会自动安装依赖，属于正常情况。请等待几分钟。

### 3. 网页打不开

请确认黑色窗口还开着，然后手动访问：

```text
http://127.0.0.1:3030
```

### 4. 想换电脑怎么办

在旧电脑备份这些文件：

```text
fengdingding-progress.json
fengdingding-progress.db*
backups/
llm-config.json
```

复制到新电脑的 QuantLife 文件夹里，再双击启动。

注意：`llm-config.json` 可能包含 API Key，只能自己保存。
